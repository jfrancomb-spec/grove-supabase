# Grove Supabase

This repository stores the Supabase backend code for Grove.

## What Goes Here

- `supabase/config.toml`
- `supabase/functions/`
- `supabase/migrations/`

## Suggested Workflow

1. Add SQL changes as migration files in `supabase/migrations/`
2. Add or update Edge Functions in `supabase/functions/`
3. Commit backend changes here before deploying them in Supabase
